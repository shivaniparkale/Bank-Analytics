# Bank-Analytics
This project focuses on analyzing bank loan data using advanced Excel features to gain insights into customer lending patterns, loan performance, and financial risk. The analysis includes data cleaning, preprocessing, KPI tracking, and interactive dashboard creation using Pivot Tables, Pivot Charts, slicers, conditional formatting, and Excel formulas. Key metrics such as total loan applications, funded amount, received amount, interest rate, and loan status were analyzed to evaluate overall business performance.

The dashboard provides insights into loan distribution by purpose, grade, employment length, home ownership, and state-wise performance. It also helps identify trends related to good and bad loans, customer repayment behavior, and financial risk assessment. This project demonstrates practical skills in data analysis, data visualization, reporting, and business intelligence using Microsoft Excel.

**Project Overview**

This project focuses on analyzing bank loan data using Microsoft Excel to understand customer lending behavior, loan performance, and financial risk. The dataset was cleaned, processed, and analyzed to generate meaningful business insights related to loan applications, funded amounts, repayment trends, and loan status. The project helps in identifying patterns that support better financial decision-making and risk assessment.

The analysis was performed using Excel dashboards and visualization techniques to monitor KPIs such as total loan applications, total funded amount, total amount received, average interest rate, and debt-to-income ratio. Interactive reports were created to make the analysis more user-friendly and insightful.
